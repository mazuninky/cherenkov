# Code of conduct

The Cherenkov project adopts the Contributor Covenant, version 2.1, as its
code of conduct.

The full text is published at:

* <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

Copy of record (with translations and an FAQ) lives at:

* <https://www.contributor-covenant.org/>

## Reporting

Reports of conduct that violates the Contributor Covenant should be sent to
**conduct@cherenkov.dev**. Reports are read by the maintainer only and are
treated as confidential.

The reporter will receive an acknowledgement within 72 hours and a
substantive response within 14 days. We follow the enforcement ladder
described in the Contributor Covenant FAQ.

## Scope

The code of conduct applies to all repository spaces (issues, pull
requests, discussions, commits, code review) and to project-controlled
public communications under the Cherenkov name.
